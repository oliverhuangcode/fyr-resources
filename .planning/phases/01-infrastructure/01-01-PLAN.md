    <action>
      Replace the default tailwind.config.ts content with one that extends the theme with MAC Learn tokens.

      The config must extend (not replace) the default theme. Add under `theme.extend.colors`:

          base: '#1C1C1C',       // bg-base
          surface: '#242424',    // bg-surface
          accent: '#F5C518',     // accent (yellow CTA)
          subtle: '#2E2E2E',     // border-subtle
        }

      And under `theme.extend.textColor` (or use the colors above with Tailwind's built-in text- prefix):
        The `text-primary` and `text-muted` tokens map to white and #888888.
        Add to colors:
          primary: '#FFFFFF',
          muted: '#888888',

      Full colors block:
        base: '#1C1C1C',
        surface: '#242424',
        accent: '#F5C518',
        subtle: '#2E2E2E',
        primary: '#FFFFFF',
        muted: '#888888',

      Usage in components:
        bg-base, bg-surface, bg-accent
        text-primary, text-muted, text-accent
        border-subtle

      Keep the content array pointing to:
        './app/**/*.{js,ts,jsx,tsx,mdx}',
        './components/**/*.{js,ts,jsx,tsx,mdx}',
